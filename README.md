#  CLI Task & Data Manager (Python)

A command-line based Task and Data Manager built with Python to demonstrate core programming concepts such as Object-Oriented Programming (OOP), file handling, and user authentication.


##  Features

-  User Signup & Login System  
-  CSV-based Data Storage (Persistent Storage)  
-  Object-Oriented Design (Classes & Inheritance)  
-  Input Validation & Error Handling  
-  Interactive Command-Line Interface (CLI)  


##  Technologies Used

- Python 3  
- Built-in Modules:
  - `csv` (data storage)
  - `os` (file handling)
